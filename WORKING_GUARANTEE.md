## 🛠️ **TROUBLESHOOTING**

### **Problem: Page won't load**
**Solution:**
```
1. Check terminal shows "Server running on http://localhost:3000"
2. Make sure port 3000 is not used by another application
3. Try http://127.0.0.1:3000/login instead
```

### **Problem: Login doesn't work**
**Solution:**
```
1. Verify credentials exactly:
   Email: admin@edusmart.edu (case-sensitive)
   Password: admin123 (case-sensitive)
2. Check browser console (F12) for error messages
3. Verify server is running
```

### **Problem: Dashboard doesn't show data**
**Solution:**
```
1. Refresh the page (F5)
2. Check if token exists in localStorage:
   - Press F12
   - Go to Application tab
   - Check localStorage
   - Should have 'token' and 'user' keys
3. If no token, login again
```
